C语言琐记二
Created2014-07-02|technologycs
一、枚举enum的使用#include <stdio.h> typedef enum _ErrorCode { SystemError = -100, MemError, FileError } ErrorCode; int main(int argc, char** argv) { ErrorCode errorCode; printf("%d\n",errorCode.MemError); return 0; } 以上输出结果为-99, 即枚举类型中的值只要不指定, 必然是逐个递增的, 而第一个值不指定则为0, 上述代码指定为-100则为-100, 以后的以此为基准逐一递增。 二、#if 1 和#if 0#if constant ...程序段1... #else ...程序段2... #endif 这里表示, 如果constant为真(非0, 随便什么数字, 只要不是0), 就编译程序段1, 否则编译程序段2。如果有#if需要顶格写。
